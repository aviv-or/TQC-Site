Quantum Epoch is coming.

Many companies, such as IBM, Google , Microsoft has released their own quantum computing simulation system or quantum prototype computer. The RSA crypto system which is the base of current cryptocurrency system will be break in quantum epoch. So it's time to be prepared for the next generation and rebuild currency base.

What is TQC?

TQC is short for Truly Quantum Cash, it's an experiment on quantum cryptography currency.

First we need to find quantum computer resistant cryptography library to replace current signature-verify system used in blockchain (include all kind of coins & so called smart contract platform) technology.

Then we need rebuild quantum compatible cryptography system which can run on quantum compute devices, to create new quantum smart contract platform.
